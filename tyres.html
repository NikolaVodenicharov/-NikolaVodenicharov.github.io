Tyres

Tyre is a web app for trading tyres. With search bar, options to calculate what bigger tyres you can use and articles for more curious one.

Description

Home page have an option to search tyres by technical specifications. All available tyres, filtered by users parameters are displayed. By clicking to the tyre of interest we can check for more details. With "buy" button users are going to see the specific iformation about the purchase. Orders are made by authenticated users, that are registered and logged in. If an user want to change his tyre size, he can check the compatibility. For more curious one there are articles section with additional useful information. Meanwhile the staff of the company have a little bit different view. With options for managing the site and processing the orders.

What is used in the project

Using C# programming language accompanied by ASP.Net Core for best performance in the back end of web app. Entity framework core is providing the ORM for smooth connection with microsoft sql server. MVC pattern is one of the most popular way of securing separation of concern in web application. Maintaining a clear detaching UI and logic. Making application easier to test, maintain and evolve. A domain driven design that is positioning the business logic and application models at the center. They do not depend of any data access of infrastructure. The dependency is inverted, implementation details depend of the core. This functionallity is achieved by abstraction and interfaces.
